Get alerts when bot dies/hangs (recommended for production):

1. Get Slack webhook URL:
   https://api.slack.com/messaging/webhooks
   
   Example: https://hooks.slack.com/services/T00/B00/XXXX

2. Set webhook URL:
   export SLACK_WEBHOOK_URL="https://hooks.slack.com/services/..."
   echo 'export SLACK_WEBHOOK_URL="...">>' ~/.bashrc

3. Setup automated monitoring:
   /tmp/setup_monitoring_cron.sh

4. Test it works:
   cd /root/hyperliquid-mm-bot-complete
   ./scripts/slack_alert.sh

5. Monitor logs:
   tail -f /var/log/mm_bot_monitor.log

Done! You'll get Slack alerts every 5 min if bot is down.
